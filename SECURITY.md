# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Paper Boy News, please report it responsibly.

**Email:** [security@paper-boy-news.com](mailto:security@paper-boy-news.com)

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours. Please do not open a public issue for security vulnerabilities.

## Supported Versions

Only the latest version on the `main` branch is supported with security updates.
